# 交易执行与复盘检查清单

> 建议打印或保存本清单，每次按阶段逐项勾选。

---

## 一、盘前检查（开盘前 30 分钟）

### 1. 系统检查
- [ ] **交易系统**：交易软件/接口运行正常，无报错
- [ ] **网络连接**：延迟稳定，无频繁断线
- [ ] **账户资金**：可用资金充足，保证金无压力
- [ ] **止损设置**：预设止损单已正确挂出并确认

### 2. 持仓检查
- [ ] **止损位**：所有持仓均已设置止损
- [ ] **触发情况**：止损位未被提前触发或误触发
- [ ] **仓位限制**：总仓位未超过系统设定上限
- [ ] **相关性**：品种间相关性未超标，避免集中风险

### 3. 市场环境检查
- [ ] **重要数据**：今日是否有重大数据/事件（若有，已制定应对方案）
- [ ] **环境适配**：当前市场环境仍适用于本交易系统（趋势/震荡等）
- [ ] **极端波动**：无熔断、极端跳水/拉升等异常情况

### 4. 心理状态检查
- [ ] **情绪状态**：情绪稳定（完成既定自检流程）
- [ ] **睡眠质量**：睡眠时间 ≥ 6 小时，精神状态良好
- [ ] **生活干扰**：无重大生活事件强烈干扰判断
- [ ] **专注力**：注意力集中，无明显分心

### 5. 当日交易计划
- [ ] **机会识别**：潜在交易机会已初步筛选
- [ ] **入场条件**：入场触发条件明确、可量化
- [ ] **仓位计算**：每笔与总仓位已按风险模型计算
- [ ] **止损方案**：价格止损/时间止损已设定
- [ ] **最坏情形**：已预想最坏情况并接受其结果

---

## 二、盘中检查（每小时一次）

- [ ] **持仓止损**：止损是否仍然有效、未被误删或修改
- [ ] **新信号**：是否出现新的有效交易信号
- [ ] **情绪波动**：是否出现明显情绪波动（愤怒、恐慌、亢奋）
- [ ] **纪律执行**：是否有违反既定交易规则的行为
- [ ] **是否调整**：是否需要调整仓位或计划（默认：不调整）

---

## 三、周末复盘（建议用时：2 小时）

### 1. 数据统计
- [ ] **本周交易笔数**
- [ ] **本周胜率**
- [ ] **本周盈亏比**
- [ ] **本周最大单笔盈利**
- [ ] **本周最大单笔亏损**
- [ ] **本周最大回撤**
- [ ] **本周连续亏损次数**

### 2. 执行质量评估
- [ ] **违反系统次数**（如：未按信号交易、随意加减仓）
- [ ] **情绪失控次数**（冲动交易、报复性交易等）
- [ ] **纪律评分**：1–10 分，主观评价本周纪律执行
- [ ] **主要问题**：列出 1–3 个本周最核心问题

### 3. 系统评估
- [ ] **系统有效性**：当前系统是否仍有正期望值迹象
- [ ] **市场环境变化**：本周市场结构是否明显变化
- [ ] **是否需要暂停**：如连续亏损 > 10 笔，考虑暂停系统
- [ ] **参数调整需求**：是否需要微调参数（谨慎评估）

### 4. 心理状态评估
- [ ] **压力水平**：1–10 分
- [ ] **信心水平**：1–10 分
- [ ] **自信状态**：是否过度自信或明显自卑
- [ ] **休息需求**：是否需要刻意安排休息/减量交易

### 5. 下周计划
- [ ] **仓位调整**：如有需要，调整风险暴露
- [ ] **重点品种**：列出下周重点关注标的
- [ ] **风险上限**：设定或更新单笔/总风险上限
- [ ] **心理准备**：为可能的波动/连续亏损做心理预期

---

## 四、月末深度复盘（建议用时：4 小时）

### 1. 绩效分析
- [ ] **月收益率**
- [ ] **月夏普比率**（如有条件计算）
- [ ] **月最大回撤**
- [ ] **月交易笔数**
- [ ] **月胜率**
- [ ] **月盈亏比**
- [ ] **与基准比较**：与指数或其他基准对比表现

### 2. 风险检查
- [ ] **单笔风险**：是否有单笔风险超出预设上限
- [ ] **总风险敞口**：整体风险暴露是否超标
- [ ] **相关性风险**：持仓间相关性是否过高
- [ ] **杠杆水平**：是否使用了过高杠杆
- [ ] **黑天鹅防护**：是否有基本防护（止损、分散、避开重大事件）

### 3. 系统健康度
- [ ] **正期望值**：系统是否仍体现正期望特征
- [ ] **样本量**：当月及累计样本量是否足够（目标：> 100 笔）
- [ ] **失效期判断**：是否可能进入系统失效期
- [ ] **环境变化**：市场结构是否发生较大变化
- [ ] **优化需求**：是否需要优化系统（谨慎、基于数据）

### 4. 心理评估
- [ ] **交易体验**：是否仍然享受交易过程
- [ ] **生活质量**：交易是否明显影响生活质量
- [ ] **结果执着**：是否过度执着于盈亏结果
- [ ] **休假需求**：是否需要安排一段时间完全离场休息

### 5. 资金管理
- [ ] **提取利润**：是否需要提取部分利润落袋
- [ ] **增加资金**：是否有必要增加交易资金
- [ ] **降低仓位**：是否需要整体降低风险暴露
- [ ] **资产配置**：整体资产配置是否合理（交易资金占比）

### 6. 学习与成长目标
- [ ] **本月最大教训**：用一两句话总结
- [ ] **需要改进的点**：列出 1–3 个具体改进方向
- [ ] **下月学习目标**：设定明确可执行的学习目标
- [ ] **阅读计划**：列出需要阅读的书籍/资料/研究报告

## 五、连续亏损应对（6.1）

### 1. 立即行动
1. **停止新开仓**：至少 24 小时内不再开新仓
2. **检查持仓止损**：确认所有持仓止损有效且合理
3. **降低仓位**：总仓位降至原来的 50%
4. **记录情绪**：写情绪日记，记录当下感受与想法

### 2. 原因分析
- [ ] **系统失效？**：样本量是否足够，是否进入失效期
- [ ] **市场环境变化？**：如趋势转为震荡等结构变化
- [ ] **执行问题？**：是否频繁违反纪律或随意操作
- [ ] **心理问题？**：是否因情绪失控导致错误决策
- [ ] **纯随机？**：是否在系统预期范围内的概率事件

### 3. 决策指引
- **若为随机/失效期内波动**：继续执行系统，但降低仓位
- **若为市场环境变化**：暂停系统，等待环境回到适用区间
- **若为执行问题**：制定并执行纪律强化训练（如打勾记录）
- **若为心理问题**：停止交易，至少休息 1 周，专注恢复状态

---

## 六、大幅回撤应对

### 1. 立即行动
1. **降低仓位**：总仓位降至原来的 50%
2. **检查止损**：确认所有止损设置合理且已生效
3. **停止新开仓**：至少 48 小时内不再开新仓
4. **冷静分析**：暂不操作，先完成原因分析

### 2. 原因排查
- [ ] **正常波动？**：是否在系统预期波动范围内
- [ ] **黑天鹅事件？**：是否为极端、低概率突发事件
- [ ] **系统失效？**：系统逻辑是否不再适应当前市场
- [ ] **执行错误？**：是否存在明显操作失误
- [ ] **杠杆过高？**：是否因过高杠杆放大损失

### 3. 应对策略
- **正常波动**：继续执行，不随意修改系统
- **黑天鹅事件**：检查防护措施是否到位，后续仍按系统执行
- **系统失效**：暂停交易系统，进行全面评估与回测
- **执行错误**：制定纠错方案，强化纪律与流程
- **杠杆过高**：将杠杆降至安全水平，重新评估风险承受能力
