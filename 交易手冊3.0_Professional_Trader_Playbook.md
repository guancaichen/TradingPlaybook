# 第一章 交易哲學
- 不預測市場，只跟隨市場。
- 保住本金優先於追求收益。
- 贏家讓利潤奔跑，輸家迅速止損。
- 永不攤平虧損。

# 第二章 市場環境
## 牛市
- 持股70~100%
## 震盪
- 持股30~60%
## 熊市
- 現金為王。

# 第三章 SEPA
- EPS成長>25%
- 營收成長>20%
- 相對強度RS高
- 突破平台放量買入

# 第四章 Livermore
- 試倉
- 金字塔加倉
- 不逆勢

- A 年度盈餘

# 第六章 ETF波段
## 趨勢判斷
- 站上200MA
- 成交量放大
- 市場風險偏好提升
## 建倉
- 首倉30%
- 確認突破加30%
- 趨勢延續加40%
## 止損
- 5%~8%

# 第七章 成長股
優先：
- AI
- 半導體
- 軟體
- 雲端
避免：
- 基本面惡化
- 下跌趨勢

# 第八章 倉位
- 單筆風險≤1%
- 最大回撤10%減倉
- 回撤15%停止交易檢討

# 第九章 規則
## 買入
- 趨勢向上
- 放量突破
## 加倉
- 僅盈利加倉
## 賣出
- 跌破止損
- 趨勢破壞
