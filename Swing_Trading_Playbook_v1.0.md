# Swing Trading Playbook v1.0

## 趨勢波段交易操作手冊

## 交易理念

我的工作不是預測市場，而是：

1.  保住本金
2.  抓住主升浪
3.  不與市場爭辯

> 市場永遠是對的。

### 我接受

-   停損
-   等待
-   空倉
-   錯過行情

### 我拒絕

-   攤平
-   賭博
-   猜底
-   猜頂
-   因為便宜而買進

------------------------------------------------------------------------

# 第一章 交易週期

-   **交易風格**：Swing Trading（波段交易）
-   **持股時間**：數天～數週（必要時數月）
-   **獲利目標**：單筆波段 5%～20%以上

分析核心：

-   宏觀／基本面：決定方向
-   技術面：決定進出場

------------------------------------------------------------------------

# 第二章 市場分析（Market First）

每天第一件事不是找股票，而是判斷市場是否適合交易。

市場環境：

-   多頭
-   空頭
-   盤整

規則：

-   60MA 向上，只考慮做多。
-   60MA 向下，不做多。
-   **Never fight the tape.**
-   **Trend is your friend.**

## 市場 → 板塊 → 個股（三層確認）

### 第一層：市場

-   美股 AI 是否仍為主線
-   NASDAQ 是否強勢
-   SOX 是否強勢
-   利率方向
-   美元方向
-   日經225 是否站穩均線
-   TOPIX 是否強於日經
-   外資是否持續買超

### 第二層：板塊

-   AI
-   半導體
-   機器人
-   Cloud
-   Software

### 第三層：個股

-   創52週新高
-   領先板塊
-   放量突破
-   強於大盤

只有 **市場 → 板塊 → 個股** 三者共振才交易。

------------------------------------------------------------------------

# 第三章 選股系統

只做主流產業：

-   AI
-   半導體
-   機器人
-   Cloud
-   Software

------------------------------------------------------------------------

# 第四章 進場系統

1.  拒絕左側交易，等待右側確認。
2.  只在突破時進場。
3.  不在盤整區進場。

買點包括：

-   平台突破
-   壓力突破
-   20MA站穩
-   趨勢反轉完成
-   回調結束再次上攻

------------------------------------------------------------------------

# 第五章 倉位管理

-   單筆最大風險：總資金 1%～2%
-   金字塔加倉
    -   50% → 100%
    -   或 20% → 40% → 70% → 100%

**永遠只加碼盈利部位，Never Average Down。**

------------------------------------------------------------------------

# 第六章 停損與出場

## 初始停損

-   前低
-   支撐位
-   均線下方

### 時間停損

3～5天沒有脫離成本區，直接離場。

## 移動停利

-   20MA／50MA跌破
-   波段低點跌破
-   123 Top
-   2B Top

讓利潤奔跑，不因害怕回吐提前賣出。

------------------------------------------------------------------------

# 第七章 風險控制

-   連續停損3筆
-   停止交易3～5天
-   重新分析市場

原則：

> 先控制風險，再追求收益。

------------------------------------------------------------------------

# 第八章 每日 Checklist

-   [ ] 大盤趨勢向上
-   [ ] 板塊為主流
-   [ ] 個股趨勢向上
-   [ ] 成交量正常
-   [ ] 有明確買點
-   [ ] 已設定止損
-   [ ] 單筆風險 ≤1%
-   [ ] 盈虧比 ≥3:1
-   [ ] 沒有 FOMO

有任何一項不符合，不交易。

------------------------------------------------------------------------

# 第九章 交易心理

交易前三問：

1.  市場適合做多嗎？
2.  我的進場理由清楚嗎？
3.  我能接受停損嗎？

只要有一個答案是 **NO**，放棄交易。

# 交易憲法

1.  市場永遠是對的。
2.  不預測市場，只對市場反應。
3.  永遠順勢交易。
4.  虧損要小，盈利要大。
5.  永遠不攤平，只加碼盈利部位。
6.  市場 \> 板塊 \> 個股。
7.  空倉也是一種倉位。
8.  每次違規都必須記錄。
9.  交易成功的標準是執行系統，而不是單筆盈虧。
10. 少交易，多等待，只做大概率交易。
